# Retail Sales Management System - Frontend

## Overview

React-based frontend application for the Retail Sales Management System. Provides an intuitive interface for searching, filtering, sorting, and viewing sales data.

## Tech Stack

- React 18
- Vite
- Axios
- CSS3

## Setup Instructions

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

The application will run on `http://localhost:3000`

3. For production build:

```bash
npm run build
```

## Features

- Search by customer name or phone number
- Multi-select filters for regions, genders, categories, tags, and payment methods
- Range filters for age and date
- Sorting by date, quantity, or customer name
- Pagination with 10 items per page
- Summary statistics display
- Responsive sidebar navigation
- Modern, clean UI design
